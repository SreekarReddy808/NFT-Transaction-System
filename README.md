# NFT Transaction System

A full stack NFT trading platform built using React, TypeScript, Spring Boot, Python and Docker.

## Tech Stack

Frontend: React + TypeScript + Bootstrap  
Backend: Java Spring Boot  
Microservice: Python Flask  
Database: MySQL / PostgreSQL  
CI/CD: Jenkins  
Containerization: Docker  
Cloud: Google Cloud Platform

## Features

- NFT Minting
- NFT Marketplace
- Transaction Tracking
- Wallet Management

## Setup

Frontend
npm install
npm start

Backend
mvn spring-boot:run

Python Validator
pip install -r requirements.txt
python nft_validator.py
